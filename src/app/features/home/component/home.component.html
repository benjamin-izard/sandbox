SANBOX HOME PAGE

The Sandbox application is a training platform to test angular features and improve web development skills.

There are several pages to test some usual features like "Employee" page for API calls.

This application will evolve over the time by adding some new fancy features and try greate CSS tricks !

It uses Angular Material for theming and components styles and makes CRUD operations with a Spring Boot backend.

It is also useful for real work project purpose such as Angular version migrations and so on...

Feel free to play with the app and analyse the code
